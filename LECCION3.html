Lección 3: Sistema de Contraseña con Teclado y LCD

Objetivos

  • Implementar un sistema de seguridad digital utilizando una placa Arduino Uno.
  • Gestionar la entrada de datos mediante un teclado matricial (Keypad 4x4).
  • Visualizar estados de validación en tiempo real a través de una pantalla LCD 16x2.
  • Configurar una alerta sonora mediante un buzzer para intentos fallidos.

Definición del Proyecto

Esta práctica consiste en la creación de un control de acceso electrónico. El usuario debe ingresar una secuencia numérica predefinida. El sistema procesa la entrada y compara los caracteres; si coinciden, se otorga acceso visual. En caso contrario, el sistema activa una alarma sonora, demostrando la integración de periféricos de entrada, procesamiento y salida.

Materiales Utilizados

  • Microcontrolador: Arduino Uno R3.
  • Entrada: Teclado Matricial 4x4 (Keypad).
  • Salida Visual: Pantalla LCD 16x2.
  • Salida Sonora: Buzzer piezoeléctrico (Alarma).
  • Componentes: Resistencia de 220Ω, potenciómetro y cables.

Conclusiones

  • Se logró la lectura correcta de una matriz de teclas mediante la programación de filas y columnas.
  • La interfaz LCD proporciona una respuesta inmediata al usuario, mejorando la experiencia de uso.
  • Se validó que el sistema bloquea el acceso de manera efectiva activando el buzzer ante claves incorrectas.

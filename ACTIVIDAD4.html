PRÁCTICA 4: ARDUINO SENSOR DE DISTANCIA

Descripción de la Actividad

En esta práctica se realizó la programación de una placa Arduino Uno para controlar un sensor ultrasónico HC-SR04. El objetivo es medir la distancia: si es mayor a 100 cm se enciende un LED, y si es menor, se activa una alarma sonora (buzzer).

Análisis de Componentes

  • Arduino Uno R3: El cerebro que procesa y ejecuta las instrucciones.
  • Sensor Ultrasónico HC-SR04: Mide la distancia enviando ondas de sonido.
  • Resistencias de 220Ω: Protegen el LED.
  • Buzzer (Piezo): Emite sonido de alerta.
  • LED: Indicador visual de estado seguro.

Código de Arduino


long duracion;
int distancia;

void setup() {
  pinMode(13, OUTPUT);
  pinMode(10, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  distancia = (duracion / 2) / 29;

  if (distancia > 100) {
    digitalWrite(13, HIGH);
    digitalWrite(10, LOW);
  } else {
    digitalWrite(13, LOW);
    digitalWrite(10, HIGH);
  }
}

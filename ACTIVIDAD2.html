Deber 2: Encendido de LEDs

Descripción Técnica

Esta práctica demuestra el control de salidas digitales mediante la placa Arduino Uno. Se implementó un circuito con 6 LEDs conectados a los pines digitales del 1 al 7. El programa utiliza estructuras de repetición para optimizar el encendido secuencial.

El Código Fuente

const int PIN_INICIO = 1;
const int PIN_FIN = 7;

void setup() {
    for(int i = PIN_INICIO; i <= PIN_FIN; i++) {
        pinMode(i, OUTPUT); // Configura pines como salida
    }
}

void loop() {
    // 1. Secuencia Ascendente con FOR
    for(int i = PIN_INICIO; i <= PIN_FIN; i++) {
        secuencialLed(i);
    }

    // 2. Secuencia Descendente con WHILE
    int j = PIN_FIN;
    while(j >= PIN_INICIO) {
        secuencialLed(j);
        j--;
    }
}

void secuencialLed(int pin) {
    digitalWrite(pin, HIGH);
    delay(500);
    digitalWrite(pin, LOW);
    delay(500);
}

Análisis de Componentes

  • Arduino Uno R3: El cerebro que procesa y ejecuta el código cargado.
  • Resistencias de 220Ω: Componentes esenciales para limitar la corriente y proteger los LEDs.
  • Estructura FOR: Utilizada para iterar de manera eficiente cuando conocemos el rango de pines.
  • Estructura WHILE: Implementada para manejar el flujo descendente bajo una condición lógica.

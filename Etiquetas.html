Diccionario de Etiquetas HTML

Etiqueta Definición Ejemplo de uso
<h1> a <h6> Define los títulos del documento. El 1 es el más importante. <h1>Título Principal</h1>
<p> Define un párrafo de texto. <p>Este es un párrafo.</p>
<a> Define un hipervínculo (enlace) a otra página. <a href="url">Enlace</a>
<ul> Define una lista desordenada (con viñetas). <ul><li>Item</li></ul>
<strong> Define texto importante (resaltado en negrita). <strong>Importante</strong>
<img> Inserta una imagen en el documento. <img src="img.jpg" alt="texto">

Importancia de las Etiquetas HTML

  • Las etiquetas HTML estructuran el contenido, facilitando su lectura.
  • Permiten a los navegadores interpretar correctamente el contenido.
  • Mejoran la accesibilidad y el SEO del sitio web.

Conclusión

  • El uso adecuado de etiquetas es esencial para sitios efectivos.
  • Comprender su función ayuda a diseñar webs más eficientes.

Diccionario de Etiquetas HTML5

Etiqueta Definición Ejemplo de uso
Contenido Básico
<h1> a <h6> Títulos del documento (H1 es el principal). <h1>Título</h1>
<p> Define un párrafo de texto. <p>Hola mundo</p>
<a> Crea un enlace a otra web o archivo. <a href="url">Link</a>
<img> Muestra una imagen (requiere src y alt). <img src="f.jpg" alt="des">
Estructura Semántica (Layout)
<header> Encabezado de la página o de una sección. <header>...</header>
<nav> Contenedor de enlaces de navegación (menús). <nav>...</nav>
<main> Contenido principal y único del documento. <main>...</main>
<footer> Pie de página (créditos, contacto). <footer>...</footer>
Listas y Contenedores
<ul> / <ol> Lista desordenada (puntos) u ordenada (números). <ul><li>Item</li></ul>
<div> Contenedor de bloque genérico para diseño. <div>...</div>
<span> Contenedor en línea para fragmentos de texto. <span>Texto</span>
Formularios e Interacción
<form> Área que contiene elementos de entrada de datos. <form>...</form>
<input> Campo de entrada (texto, email, password, etc). <input type="text">
<button> Botón clickeable para ejecutar acciones. <button>Enviar</button>

Importancia de las Etiquetas HTML

  • Semántica: Ayudan a los buscadores (Google) a entender qué partes son importantes.
  • Accesibilidad: Permiten que lectores de pantalla para personas con discapacidad funcionen bien.
  • Organización: Facilitan el mantenimiento del código a largo plazo.

Conclusión

  • El dominio de estas etiquetas es el primer paso para ser un desarrollador Web Pro.
  • Combinar etiquetas semánticas con CSS permite crear sitios modernos y rápidos.

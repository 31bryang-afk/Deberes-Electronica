Examen Teórico: Documentación del Conversor

1. Descripción General

Este programa en Python permite al usuario convertir números entre los sistemas Binario, Octal, Decimal y Hexadecimal mediante un menú interactivo de selección (1-4).

2. Descripción de Componentes (Lo que pide el examen)

Variables y Diccionarios: Utilizamos un diccionario llamado bases para mapear las opciones del menú (1, 2, 3, 4) con sus valores reales (2, 8, 10, 16).
Función int(numero, base): Es el motor del programa. Convierte cualquier entrada a sistema decimal para usarlo como puente de conversión.
Condicionales (If/Elif): Permiten que el programa decida qué función de salida usar (bin, oct, hex) según la elección del usuario.

3. Código Fuente

    print("--- BIENVENIDO ---")
    # primero colocamos esto en la terminal = python conversor.py
    
    # 1. Pedir el número primero
    numero = input("Ingresa el número que quieres convertir: ")

    # 2. Menú para la base de ORIGEN
    print("\n¿En qué sistema está ese número?")
    print("1. Binario")
    print("2. Octal")
    print("3. Decimal")
    print("4. Hexadecimal")
    opcion_origen = input("Elige una opción (1-4): ")

    # Diccionario para convertir opciones en bases numéricas
    bases = {"1": 2, "2": 8, "3": 10, "4": 16}
    base_actual = bases.get(opcion_origen)

    # 3. Menú para la base de DESTINO
    print("\n¿A qué sistema lo quieres convertir?")
    print("1. Binario")
    print("2. Octal")
    print("3. Decimal")
    print("4. Hexadecimal")
    opcion_destino = input("Elige una opción (1-4): ")
    base_deseada = bases.get(opcion_destino)

    if not base_actual or not base_deseada:
        print("❌ Opción no válida.")
        return

    try:
        # Lógica de conversión (el puente decimal)
        decimal = int(numero, base_actual)

        if base_deseada == 2:
            resultado = bin(decimal)[2:]
        elif base_deseada == 8:
            resultado = oct(decimal)[2:]
        elif base_deseada == 16:
            resultado = hex(decimal)[2:].upper()
        else:
            resultado = str(decimal)

        print(f"\n✅ El número {numero} convertido es: {resultado}")

    except ValueError:
        print("\n❌ Error: El número no coincide con el sistema de origen seleccionado.")
